# Telegram-бот для ростерії

Бот збирає клієнтів, які самі підписались (написали /start), розподіляє їх по
сегментах з власним щотижневим розкладом розсилки, і дозволяє миттєво
розіслати повідомлення всім (наприклад, зміна прайсу чи акція).

## Що вміє бот

- Клієнт пише боту `/start` → потрапляє у базу підписників.
- Вам (адміну) приходить сповіщення про нового підписника з кнопками —
  одним тапом призначаєте йому сегмент (наприклад "роздріб", "опт", "кафе").
- Кожен сегмент має свій день і час щотижневої розсилки та своє повідомлення.
- Окрема команда `/broadcast` — надсилає повідомлення ОДРАЗУ всім підписникам,
  незалежно від сегмента й розкладу (для термінових новин, зміни цін, акцій).

## Крок 1. Створити бота в Telegram

1. Відкрийте `@BotFather` в Telegram.
2. Команда `/newbot`, дайте назву й юзернейм (має закінчуватись на `bot`).
3. Отримаєте токен виду `123456:ABC-DEF...` — збережіть його.

## Крок 2. Дізнатись свій ADMIN ID (і колег, якщо потрібно)

1. Напишіть боту `@userinfobot` будь-яке повідомлення.
2. Він покаже ваш числовий Telegram ID.
3. Якщо хочете, щоб сповіщення про нових клієнтів і керування ботом було
   доступне ще й колегам — попросіть кожного з них так само написати
   `@userinfobot` і скинути вам своє число.

## Крок 3. Налаштувати проєкт

1. Скопіюйте `.env.example` у `.env`.
2. Впишіть туди свій `BOT_TOKEN`.
3. У `ADMIN_IDS` впишіть один ID (тільки ви) або кілька через кому без
   пробілів (ви і колеги), наприклад:
   ```
   ADMIN_IDS=111111111,222222222,333333333
   ```
   Кожен із цих людей зможе отримувати сповіщення про нових підписників і
   керувати ботом (сегменти, розклад, розсилки) однаково — прав "головного"
   і "другорядного" адміна немає, усі рівні.

## Важливо: цей бот — лише для нагадувань і новин

Бот навмисно **не приймає** повідомлення від клієнтів як замовлення чи
звернення — він тільки надсилає нагадування за розкладом і термінові
розсилки. Якщо клієнт відповість щось у чаті з ботом, бот це просто
проігнорує.

Для живого спілкування з клієнтами (питання по замовленню тощо)
рекомендується вказувати прямо в тексті розсилки контакт для зв'язку
(юзернейм або номер телефону), щоб клієнт писав вам чи колезі напряму,
в обхід бота — так зберігається звичайна особиста переписка з історією.
Приклад тексту для `/setmsg`:
```
/setmsg roztil Нагадуємо про новинки цього тижня ☕️ Питання по замовленню — пишіть @vasha_nazva
```

## Крок 4. Запуск (варіант без сервера — Railway.app, безкоштовно для старту)

1. Зареєструйтесь на [railway.app](https://railway.app) через GitHub.
2. Завантажте ці файли (`bot.py`, `requirements.txt`, `.env.example`) у новий
   GitHub-репозиторій.
3. У Railway: New Project → Deploy from GitHub repo → оберіть репозиторій.
4. У розділі Variables додайте `BOT_TOKEN` і `ADMIN_IDS` (значення зі свого `.env`).
5. У Settings → Start Command вкажіть: `python bot.py`
6. Railway сам встановить залежності з `requirements.txt` і запустить бота.
   Він працюватиме постійно, навіть коли ваш комп'ютер вимкнено.

Альтернатива — [PythonAnywhere](https://www.pythonanywhere.com) (безкоштовний
план, трохи складніше налаштувати процес, що працює цілодобово) — якщо
знадобиться, підкажу покроково саме під цей варіант.

## Крок 5. Використання

**Створити сегменти клієнтів:**
```
/addsegment roztil
/addsegment opt
/addsegment cafe
```

**Задати розклад для кожного (день: mon/tue/wed/thu/fri/sat/sun):**
```
/setschedule roztil mon 10:00
/setschedule opt wed 09:00
/setschedule cafe fri 15:00
```

**Задати текст, який автоматично піде за розкладом:**
```
/setmsg roztil Доброго ранку! Нагадуємо про наші новинки цього тижня ☕️
/setmsg opt Прайс на цей тиждень без змін, поставки за графіком.
```
Текст можна оновлювати будь-коли — наступного разу піде вже новий.

**Призначити клієнту сегмент:** відбувається автоматично кнопками, коли він
пише `/start` — сповіщення прилітає вам у чат з ботом.

**Подивитись усі сегменти й розклади:**
```
/segments
```

**Подивитись усіх активних підписників:**
```
/clients
```

**Терміново розіслати всім (акція, зміна цін):**
```
/broadcast Увага! З понеділка нова партія кави Ефіопія Гуджі вже в продажу 🎉
```

## Важливо

- Бот може писати тільки тим, хто сам натиснув `/start` — це обмеження
  Telegram проти спаму, і водночас захист вашого акаунту.
  Розповсюджуйте посилання на бота (QR-код на упаковці, сторіз, чек тощо),
  щоб клієнти самі підписувались.
- Клієнт будь-коли може відписатись командою `/stop`.
- Часовий пояс розкладу — Europe/Kyiv, вшито в код.
